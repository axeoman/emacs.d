* Macros
- <F3> or C-x (  - Start macro
- <F4> or C-x )  - end macro
- <F4> or C-x e  - execute macro
- C-u num C-x e  - execute macro `num` times
- C-x C-k n      - name macro
- name-last-kbd-macro
- insert-kbd-macro
